Abstract

Understanding public perception is crucial for successful implementation of government policies. This study employs established Natural Language Processing (NLP) techniques to gauge public sentiment of government policies in Indian context and identify key themes related to policies in the South Karnataka region of India. We explored traditional machine learning models (SVM, Decision Tree, NaÃ¯ve Bayes, KNN, Random Forest) along with deep learning techniques (DistilBERT) for uncovering public opinion on government policies under very low training data conditions. Latent themes in public discourse were identified using K-Means Clustering and BERTopic.

Benchmarking Machine Learning Techniques in Under-Resourced Contexts

Analysis of Public Perceptions of Government Policies from South Karnataka Reddit Discourse

Research Objective

Understanding public perception is crucial for successful implementation of government policies. This study aims to employ established Natural Language Processing (NLP) techniques to gauge public sentiment regarding government policies in the Indian context and identify key themes related to policies in the South Karnataka region.

Methodology

We explored a comprehensive range of machine learning approaches:

Traditional ML Models

  • Support Vector Machine (SVM) - Robust classifier for sentiment analysis
  • Decision Tree - Interpretable classification approach
  • NaÃ¯ve Bayes - Probabilistic text classification
  • K-Nearest Neighbors (KNN) - Instance-based learning
  • Random Forest - Ensemble method for improved accuracy

Deep Learning

  • DistilBERT - Lightweight transformer model for sentiment classification

Topic Modeling

  • K-Means Clustering - Identifying discourse patterns
  • BERTopic - Advanced topic extraction using transformers

Dataset & Challenges

Low-Resource Context

Working with limited labeled training data for South Karnataka region

Reddit Discourse

Analyzing 500+ posts from regional subreddits discussing government policies

Regional Context

Understanding sentiment in Indian policy context with regional nuances

Key Findings

Model Performance

Benchmarked 6 ML algorithms under very low training data conditions, providing insights into model robustness

Sentiment Patterns

Successfully identified public sentiment trends toward government policies in South Karnataka

Thematic Analysis

Uncovered latent themes in public discourse using clustering techniques

Policy Insights

Provided actionable insights for policy makers on public perception

Research Impact

This study provides a methodological framework for analyzing public sentiment in under-resourced linguistic contexts. The findings demonstrate that effective sentiment analysis is possible even with limited training data, offering valuable guidance for policy makers seeking to understand public opinion on government initiatives.
